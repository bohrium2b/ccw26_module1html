Property Description Example
color Sets the color of text This text is blue
background-color Sets the background color of an element This text has a yellow background
font-family Sets the font of text This text is in Courier New font
font-size Sets the size of text This text is 24 pixels in size
Optional Properties
font-weight Sets the weight (boldness) of text This text is bold
font-style Sets the style of text (e.g., italic) This text is italic
text-decoration Adds decoration to text (e.g., underline) This text is underlined
